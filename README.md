# Crow Login - Email Verification

A beautiful Next.js email verification page with OTP input.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your images to the `public` folder:
   - `public/bg.jpg` - Background image
   - `public/logo.png` - Crow logo

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## Features

✅ 6-digit OTP input with auto-focus
✅ Paste support for OTP
✅ Countdown timer with resend option
✅ Glass-morphism card design
✅ Smooth animations
✅ Fully responsive
✅ Keyboard navigation
